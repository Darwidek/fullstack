RUN THIS LOCAL PROJECT!!!

IF YOU DOWNLOAD ZIP. Extract files and open root folder: fullstack and Run in port localhost:4200. 

IF YOU RUN THIS IN CONSOLE (GIT BASH RECOMMENDED). Please, run git clone -b main https://github.com/Darwidek/fullstack.git an cd in root folder fullstack and run npm install, finally you can run ng serve  in port localhost:4200. 

¡¡¡¡IMPORTANT!!!. This application requires Angular CLI and NPM be installed.


This project is the frontend part, you must run the joint backend project in FullstSpringboot_angular_crud in repository at the same time. (please read the corresponding Readme).
